# MVP Roadmap

## Fase 1 (4-6 semanas)
- Auth + roles.
- Onboarding paciente con intake y descarte de riesgo.
- Perfil profesional publico.
- Slots + reserva + cancelacion (24h).
- Checkout Stripe para paquetes.
- Creditos de sesion y consumo por booking.
- Chat texto basico.
- Video por sesion (Daily).

## Fase 2
- Sincronizacion Google Calendar opcional.
- Notificaciones push/email completas.
- Dashboard de ingresos mas profundo.
- IA auditor de texto en produccion con revision humana.

## Fase 3
- IA de audio/transcripcion con consentimiento explicito.
- Automatizacion de insights clinicos asistidos.
- Analitica avanzada de conversion/retencion.
